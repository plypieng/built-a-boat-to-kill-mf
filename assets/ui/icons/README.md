First-pass HUD and unlock icons for the current hangar/run loop.

These are local SVG assets so the project gets usable art immediately even though no image API key is configured in this shell right now.

- `storm blue` `#183642`
- `chart cream` `#EFE3C2`
- `oxidized teal` `#3E6F73`
- `sea-glass green` `#73BDA8`

- `flare red-rust` `#A84E39`

Suggested uses:
- `helm.svg` for helm ownership or piloting affordances

- `cargo.svg` for cargo capacity and secured salvage
- `salvage.svg` for salvage reward and salvage cache callouts

- `reinforced-hull.svg` for the unlock yard
